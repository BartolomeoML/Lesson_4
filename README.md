В качестве основного источника данных выбран датасет б/у автомобилей с платформы Craigslist: https://www.kaggle.com/datasets/austinreese/craigslist-carstrucks-data (vehicles.csv - 1.45 GB)

Из полного набора на 26 колонок собран датасет на 13 колонок: region,price,year,manufacturer,model,condition,cylinders,fuel,odometer,transmission,type,paint_color,state

Ссылка на файл датасета:
https://raw.githubusercontent.com/BartolomeoML/Lesson_4/refs/heads/main/dataset_used_vehicles.csv (37 Mb)

Цель: тренировка подготовки датасета для обучения.
Предполагаемая бизнес задача: предсказание стоимости б/у автомобилей на рынке США.

В ходе подготовки датасета дополнительно исключены невлияющие на ценообразование колонки, пустые поля заполнены медианными и наиболее часто встречающимися значениями, в зависимости от признака. Очищены и приведены к единому виду значения ключевой колонки с названиями моделей авто, но т.к. основная цель это обучение, то 13% строк с излишней вариативностью написания названий моделей были просто исключены. После чего проведена визуализация полученного датасета.
